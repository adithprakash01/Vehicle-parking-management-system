# Vehicle Parking Management System

This project is a simple Java-based GUI application developed using Swing that helps manage vehicle parking efficiently. It allows users to record vehicle entry, calculate parking fees during exit, and manage records in a structured way.
The Vehicle Parking Management System is designed to simplify the process of handling parking entries and exits. It provides an easy-to-use interface where users can:
Add vehicle details when entering, Record exit date, Automatically calculate parking fees, Delete records when needed.

This project helped in understanding Java GUI development, event handling, and basic data management using ArrayList.

# Features
 Add vehicle entry with details (Vehicle No, Make, Model, Slot)
 Automatic entry time generation
 Exit date input with validation
 Parking fee calculation based on number of days
 Delete vehicle records
 Clean and user-friendly GUI design
 Table display for all parking records
# Technologies Used
Java
Swing (GUI)
AWT
ArrayList (for data storage)
# Project Structure
Vehicle.java – Defines vehicle details and structure
VehicleManager.java – Handles storing, adding, and deleting vehicles
VehicleGUI.java – Main GUI application with all functionalities
#  How It Works
Entry Process
Enter vehicle details

Exit Process

Fee is calculated based on number of days
Delete Record
